Тестовая страница

Это демонстрационная страница, созданная для того, чтобы показать, как работает наш генератор.

Как это устроено?

Всё очень просто. Вы создаете новый файл .md в папке generator/src/pages/blog/ и пишете текст. Astro автоматически делает несколько вещей:

  1. Берет ваш текст.
  2. Пропускает его через файл BaseLayout.astro, добавляя вашу шапку, подвал и все скрипты.
  3. Генерирует готовый к публикации на GitHub Pages HTML-файл.

Пример кода:

<div class="card">
    <p>Это пример вставки кода прямо в Markdown!</p>
</div>

[!TIP] Обратите внимание, что вам даже не нужно прописывать HTML-теги для базовых вещей (заголовки, списки, ссылки). Маркдаун сделает это за вас.
